# How to set DoH on Vivaldi

<StepContainer>

### Upgrade your Vivaldi to latest version

Make sure you have installed latest Vivaldi, or you can download [here](https://vivaldi.com/download/).

### Go to Settings

Input `chrome://settings/security` in address bar, it will automatic redirect to `vivaldi://settings/security`, however, you can't use this URL directly.

### Set DNS.SB DoH Server

On `Advanced` section, enable `Use secure DNS`, choose `with Custom` and input:

```
https://doh.dns.sb/dns-query
```
