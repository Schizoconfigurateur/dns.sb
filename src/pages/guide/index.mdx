# How to use DNS.SB

## DNS.SB Name Servers

### IPv4

- `185.222.222.222`
- `45.11.45.11`

### IPv6

- `2a09::`
- `2a11::`

You may need IPv6 with full address in some old devices:

- `2a09:0000:0000:0000:0000:0000:0000:0000`
- `2a11:0000:0000:0000:0000:0000:0000:0000`
